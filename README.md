Banking Data Analysis (SQL Project)
Project Description
This project analyzes banking customer and transaction data using SQL to understand financial activity and customer behavior.

Tools Used
- MySQL
- SQL

Dataset Information
The dataset includes customer information, account details, and transaction records.

Key SQL Operations
- Table joins using JOIN
- Data filtering using WHERE
- Aggregation using SUM() and COUNT()
- Grouping using GROUP BY


Insights Generated
- Customer transaction patterns
- Account activity summary
- Financial data analysis

Conclusion
This project shows how SQL can be used to analyze banking datasets and understand financial behavior.
